🔓 PDF Password Remover (Python)

This tool removes password from PDF using Python.

📌 Install:

pip install pypdf

💻 Code:

from pypdf import PdfReader, PdfWriter

reader = PdfReader("protected.pdf")
reader.decrypt("your_password")

writer = PdfWriter()

for page in reader.pages:
    writer.add_page(page)

with open("unlocked.pdf", "wb") as f:
    writer.write(f)
